Espresso Music Bot

Privacy Policy

Ultimo aggiornamento: 24/05/2025

1. Dati raccolti
Il Bot non raccoglie né memorizza dati personali come username, email o indirizzi IP.
Potrebbero essere temporaneamente usati:

  • ID Discord degli utenti (per eseguire i comandi)
  • Informazioni sul canale vocale (per connettersi)
  • Contenuti dei messaggi (per riconoscere i comandi)

2. Utilizzo dei dati
I dati sono usati solo per eseguire comandi, non vengono memorizzati o condivisi.

3. Servizi di terze parti
Il Bot utilizza:

  • Discord API
  • YouTube tramite yt-dlp
  • Spotify API (solo per recuperare metadati)

4. Conservazione dati
Tutti i dati vengono eliminati subito dopo l'esecuzione del comando.

5. Diritti degli utenti
Gli utenti possono rimuovere il bot dal server in qualsiasi momento.

6. Contatti
Per domande: federico.reginato@example.com
